# North Health GPT

North Health GPT is a Hausa-language, voice-first AI system designed to support early health risk recognition and guide timely care-seeking in Northern Nigeria.

## Problem

Northern Nigeria faces a high burden of preventable maternal, newborn, and infectious disease deaths. A key driver is delayed care-seeking due to lack of accessible health information, especially among low-literacy Hausa-speaking populations.

## Solution

North Health GPT enables users to describe symptoms in Hausa using voice. The system processes this input using AI-based health logic and provides clear guidance along with referral to the nearest appropriate healthcare facility.

## Key Features

- Hausa voice input and output (no literacy required)
- AI-based symptom understanding and risk classification
- WHO-aligned health guidance logic
- Condition-specific healthcare facility referral
- Designed for low-end devices and low connectivity environments

## Technology Stack

- AI models for language understanding and response generation
- Hausa speech recognition (ASR)
- Hausa text-to-speech (TTS)
- Android mobile application
- Structured healthcare facility database

## Current Status

The project is in early development. Core system design and architecture are defined. Development of voice models, mobile application, and pilot deployment is planned for Phase 1.

## Roadmap

- Build voice interface and AI integration
- Train and refine Hausa speech models
- Develop Android application
- Conduct pilot deployment in Kano State
- Evaluate and scale across Northern Nigeria
